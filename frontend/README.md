# Frontend

Frontend statico (GitHub Pages) per progetto e-commerce.

## Struttura

```text
frontend/
	assets/
		css/style.css
		js/
			common.js
			config.js
			app.js
			search.js
			product.js
			cart.js
			orders.js
			login.js
			signup.js
			admin.js
	index.html
	search.html
	product.html
	cart.html
	orders.html
	login.html
	signup.html
	admin.html
```

## Setup rapido

1. Aggiorna `assets/js/config.js` con URL API backend Render.
2. Apri `index.html` in locale oppure pubblica la cartella `frontend` su GitHub Pages.

La guida completa di setup e deploy si trova nel README in root del repository.

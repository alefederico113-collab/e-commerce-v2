# Backend

Backend Express per il progetto e-commerce.

## Struttura

```text
backend/
	database/
		schema.sql
	src/
		server.js
	.env.example
	package.json
```

## Avvio rapido

```bash
npm install
cp .env.example .env
npm start
```

Compila i valori Supabase dentro `.env` prima di avviare il server.

Per documentazione completa (deploy Render + GitHub Pages), leggi il README in root del repository.
